# 📊 Student Performance Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on a student performance dataset using Python.

## Objectives
- Analyze student performance trends
- Find factors affecting scores
- Extract useful insights from data

## Features Implemented
✔ Data Cleaning  
✔ Feature Engineering  
✔ GroupBy & Aggregations  
✔ Pivot Tables  
✔ Data Visualization  
✔ Insight Extraction

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

## Sample Analysis Performed
- Average scores by gender
- Study time vs performance
- Subject-wise comparisons
- Correlation analysis

## Project Structure
```bash
Student-Performance-Analysis/
│── data/
│── notebooks/
│── images/
│── README.md
```

## Key Insights
- Students with higher study time scored better.
- Some subjects showed stronger correlations.
- Visualizations helped identify performance patterns.
